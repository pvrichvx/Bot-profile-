# Discord Profile Bot

บอทสำหรับสร้างโปรไฟล์ผู้ใช้แบบถามตอบใน Discord  
เมื่อผู้ใช้ใช้คำสั่ง `-pf` ในห้องที่กำหนด บอทจะถามคำถาม แล้วโพสต์โปรไฟล์ไปยังช่องที่ตั้งไว้

## วิธีรัน (สำหรับ Railway)
1. ตรวจสอบให้มีไฟล์:
   - `main.py`
   - `requirements.txt`
   - `Procfile`
2. ตั้งค่า Environment Variable:
   - `DISCORD_BOT_TOKEN=...`
